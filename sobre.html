Sobre a Aplicação

Informações

Esta aplicação web foi desenvolvida para gerenciar tarefas, perfis de usuários e configurações personalizadas.
O objetivo é facilitar o controle de atividades diárias, permitindo ao usuário cadastrar, visualizar e editar suas tarefas.
Além disso, oferece opções de personalização e gerenciamento de perfil.

Tecnologias Utilizadas

  • ASP.NET Core MVC
  • Entity Framework Core
  • MySQL
  • HTML, CSS e JavaScript
